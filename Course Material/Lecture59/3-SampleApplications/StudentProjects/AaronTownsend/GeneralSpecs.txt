GENERAL SPECIFICATIONS
======================

About:
This is my major assignment submission for my computer science class. It is a program which displays a list of televisions, speaker setups, receiver and peripherals in seperate tabs. This allows a user to analyze which componenets of a home theatre setup are best for them based on various parameters. Custom display lists can be created and all data files can be modified. Sorting can be done by clicking on the column headers when a data file is loaded. Deleting can be accomplished by clicking on data and hitting the delete button. While editing of individual entries can be accomplished by clicking on data and the going to File>Edit Data File.

Purpose of Program:
To make planning out a home theatre setup easier.
